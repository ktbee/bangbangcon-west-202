Core Rope Memory:
A How-To Guide

Load component with magnetic iron cores
Pass needle loaded with wire through cores
Machine will advance to next location
Strip insulation from wire and solder into place
Remove nylon pins and press and fold wires into assembly
Memory is tested with an electrical machine that uses paper tape to run its program
Finally, "pot" the memory by coating it in plastic
